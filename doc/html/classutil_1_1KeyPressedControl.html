Detailed Description

KeyPressedControl ist eine Hilfsklasse zur Zwischenspeicherung der gedrückten Tasten (keys). Zielsetzung ist es, festzustellen, ob zu jedem Tastendruck (note on) ein korrespondierender Tastenhub (note off) registriert wurde.

  • setKeyPressed(int key) setzt den Wert für die Taste key auf true.
  • isKeyPressed(int key) liefert den Wert die die Taste key.
  • clearKeyPressed(int key) setzt den Wert für die Taste key zurück.
  • clear() setzt alle Werte zurück.
