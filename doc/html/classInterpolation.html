Calculates the logarithm() of the linear mapping of the input value from interval [s1,e1] to interval [s2,e2].

  • The logarithmic result allows for a more fine grained control of the higher end of the interval.
  • The result should be normalized by dividing the result by log(e2-s2).
Returns
The log() of the linear mapping.
